# BboxPulse ⚡

**BboxPulse** est un dashboard moderne et léger conçu pour monitorer en temps réel les débits, la consommation réseau et les équipements connectés d'une BBox Bouygues Telecom. 
Propulsé par une stack Docker / Redis (ou Valkey), il offre une interface premium sombre avec des graphiques historiques détaillés.
N'étant pas développeur à la base et ayant besoin d'un outil introuvable ailleurs pour mon usage personnel, j'ai conçu ce projet à l'aide d'**Antigravity et Gemini** pour simplifier la supervision réseau domestique.

*Pour plus de détails sur le déploiement et l'utilisation, consultez le [README.md](README.md).*
